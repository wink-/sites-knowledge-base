# Inspiration

This directory holds content that sparked ideas — YouTube videos, articles, podcasts, etc.

## How it works

1. Give Hermes a YouTube URL (or any link) and say what it made you think about
2. Hermes fetches the transcript, summarizes it, and saves it here as a markdown file
3. The file includes the source, summary, and any content ideas it generated

## File format

Each file is named with the date and a short slug:

```
2025-07-15-off-grid-solar-build.md
```

Inside:
- Source URL
- Summary of what was covered
- What it made you think about (your context)
- Actionable ideas for your sites

## Using these later

When planning new content, an agent can scan this directory for relevant inspiration and turn ideas into drafts.
