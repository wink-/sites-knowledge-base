## EG4 GridBoss

The GridBoss is a micro-grid main disconnect / transfer switch. Key specs from the manual:

- **Model:** MI-200-2P-HYB-AW-03
- **Nominal AC Voltage:** 120/240 VAC (L1/L2/N)
- **Max Current:** 200A (grid), 125A (inverter port)
- Supports hybrid inverters, generator input, smart loads, AC coupling
- Remote management via EG4 Monitor Center / mobile app
- Manual bypass mode available
